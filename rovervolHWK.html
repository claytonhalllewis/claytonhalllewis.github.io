Explanation

  • Display shows a block with a periodically changing hole,
  • embedded in a four dimensional space.
  • Keyboard commands move a rover around in the 4D space...
  • ... you see the view from the rover.
  • The i-k keys move in time, so pressing i repeatedly
  • shows the time evolution of the scene.
  • Pressing r will reset the view.
  • Pressing the e key views the y-t plane along the original z axis,
  • so that time is now shown along the x axis.
  • This renders the time evolution of the cross section of the hole
  • as a static display.
  • Use the t-g keys to move in x,
  • ... y-h in y,
  • ... u-j in z.
